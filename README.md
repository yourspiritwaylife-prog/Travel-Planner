# 🧭 Travel-Planner

Telegram-бот, який складає детальний план подорожі по днях: куди піти,
де поїсти, як дістатись — і малює гарні **картки-картинки** на кожен день,
які зручно зберігати офлайн.

Працює на **безкоштовних** технологіях (API-ключі за гроші — не обовʼязкові).

---

## 🧩 Як це влаштовано (простими словами)

```
Користувач у Telegram
        │  (місто, дні, інтереси, бюджет)
        ▼
   «Мозок» (LLM)  ── складає план по днях
        │
        ▼
 Безкоштовні дані  ── реальні координати + фото (OpenStreetMap, Wikipedia)
        │
        ▼
  Генератор карток ── красива PNG-картинка на кожен день
        │
        ▼
   Telegram надсилає картки користувачу
```

**«Мозок» — змінний.** Сьогодні це безкоштовний Gemini, завтра — твій
Hermes на DigitalOcean. Міняється одним рядком у `.env`, код не чіпаємо.

## 🚀 Запуск — крок за кроком (Windows)

### 0. Встановити Python
Якщо ще немає — постав **Python 3.11+** з python.org
(під час встановлення постав галочку *Add Python to PATH*).

### 1. Встановити залежності
У папці проєкту відкрий PowerShell і виконай:

```powershell
pip install -r requirements.txt
playwright install chromium
```
> `playwright install chromium` — це «браузер», яким малюються картки. Обовʼязково.

### 2. Спочатку перевір демо (без ключів!)
```powershell
python demo.py
```
З'являться картинки в папці `output/`. Відкрий — переконайся, що картки гарні.
**Якщо це працює — половина справи зроблена.** ✅

### 3. Створити Telegram-бота (безкоштовно, 2 хв)
1. У Telegram знайди **@BotFather**
2. Напиши йому `/newbot`, дай назву
3. Він видасть **токен** (довгий рядок) — скопіюй

### 4. Взяти безкоштовний ключ Gemini (5 хв)
1. Зайди на https://aistudio.google.com/app/apikey
2. Натисни *Create API key*, скопіюй

### 5. Заповнити `.env`
Скопіюй файл `.env.example` у `.env` і встав свої значення:

```powershell
Copy-Item .env.example .env
notepad .env
```
Мінімум треба заповнити:
```
TELEGRAM_BOT_TOKEN = (токен від BotFather)
BRAIN = gemini
GEMINI_API_KEY = (ключ від Google AI Studio)
```

### 6. Запустити бота
```powershell
python main.py
```
Заходь у свого бота в Telegram, тисни **/start** — і вперед! 🎉

## 🔌 Підключення Hermes (пізніше)

Сервер DigitalOcean (`144.126.206.226`) навмисно закритий від інтернету —
назовні відкрито лише SSH (порт 22). Hermes працює **всередині** сервера й
не виглядає в мережу (це правильно з точки зору безпеки).

Тому найкращий спосіб зʼєднати їх — **запустити цей бот на тому ж сервері**,
поруч із Hermes. Тоді вони спілкуються напряму «через стіну», нічого не
відкриваючи назовні. У `.env` буде приблизно так:
```
BRAIN = hermes
HERMES_URL = http://127.0.0.1:<порт_Hermes>/...
HERMES_API_KEY = (якщо є)
```
Порт і формат Hermes знайдемо під час розгортання на сервері (зайдемо по SSH).
За потреби підправимо `planner/brain/hermes_brain.py`. Решта бота не змінюється.
